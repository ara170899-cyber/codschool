## Модуль 0: Первый запуск

### Урок 1: hello
**Название:** Hello, World!

**Теория:**
Каждый программист начинает с вывода текста на экран.
В Python для этого есть функция `print()`:
```python
print("Hello, World!")
```
`print()` выводит текст в терминал.

Правила:
- Текст оборачивается в кавычки: `"текст"` или `'текст'`
- Каждый `print()` — новая строка
- Можно вызывать `print()` несколько раз

**Задание:** Выведите две строки:
```
Hello, World!
Я учусь программировать!
```

**Начальный код:** `# Ваша первая программа`

**Тесты:**
| input | expected |
|-------|----------|
| (пусто) | Hello, World!\nЯ учусь программировать! |

**Подсказки:**
1. Используйте `print()` два раза — для каждой строки
2. Первый: `print("Hello, World!")`
3. Решение: `print("Hello, World!")\nprint("Я учусь программировать!")`

---

### Урок 2: input
**Название:** Ввод данных

**Теория:**
Функция `input()` читает текст от пользователя:
```python
name = input()
print("Привет,", name)
```
Как работает:
1. `input()` ждёт ввод пользователя
2. Результат сохраняется в переменную
3. Переменную можно использовать в `print()`

f-строки (удобный формат):
```python
name = input()
print(f"Привет, {name}!")
```
Внутри `f"..."` можно вставлять переменные в `{фигурных скобках}`.

**Задание:** Прочитайте имя и выведите: `Привет, {имя}! Добро пожаловать!`

**Тесты:**
| input | expected |
|-------|----------|
| Анна | Привет, Анна! Добро пожаловать! |
| Мир | Привет, Мир! Добро пожаловать! |

**Подсказки:**
1. Начните с: `name = input()`
2. Используйте f-строку: `print(f"Привет, {name}! ...")`
3. Решение: `name = input()\nprint(f"Привет, {name}! Добро пожаловать!")`

## Модуль 1: Переменные и типы

### Урок 3: variables
**Название:** Переменные и числа

**Теория:**
Переменная — имя для значения:
```python
age = 25        # int — целое число
height = 1.75   # float — дробное число
name = "Алиса"  # str — строка
```

Операции с числами: `+`, `-`, `*`, `/`, `//` (целочисленное деление), `%` (остаток)

Преобразование input → число:
```python
x = int(input())   # строку в целое число
```

**Задание:** Конвертер температур: прочитайте °C, переведите в °F по формуле `F = C × 9/5 + 32`, выведите целое число.

**Тесты:**
| input | expected |
|-------|----------|
| 100 | 212 |
| 0 | 32 |
| -40 | -40 |

**Подсказки:**
1. `c = int(input())`
2. `f = c * 9 // 5 + 32`
3. Решение: `c = int(input())\nf = c * 9 // 5 + 32\nprint(f)`

---

### Урок 4: strings
**Название:** Строки и форматирование

**Теория:**
Строки — текст в кавычках. Операции: `len(s)`, `s.upper()`, `s[0]` (первый символ), конкатенация `s + "!"`.

f-строки для форматирования:
```python
name = "Мир"
print(f"{name}, возраст: {25}")
```

**Задание:** Прочитайте имя и фамилию (2 строки). Выведите: `{Фамилия} {И}.` (фамилия + первая буква имени с точкой)

Пример: Анна / Иванова → `Иванова А.`

**Тесты:**
| input | expected |
|-------|----------|
| Анна\nИванова | Иванова А. |
| Пётр\nСидоров | Сидоров П. |

**Подсказки:**
1. Прочитайте: `name = input()` и `surname = input()`
2. Первая буква: `name[0]`
3. Решение: `name = input()\nsurname = input()\nprint(f"{surname} {name[0]}.")`

## Модуль 2: Условия

### Урок 5: ifelse
**Название:** Условия if/else

**Теория:**
```python
age = int(input())
if age >= 18:
    print("Доступ разрешён")
else:
    print("Доступ запрещён")
```
Сравнения: `==`, `!=`, `>`, `<`, `>=`, `<=`
elif — несколько веток. Логические: `and`, `or`, `not`

**Задание:** Калькулятор доставки по весу (кг): ≤1 → 100, ≤5 → 300, ≤20 → 500, >20 → 1000

**Тесты:**
| input | expected |
|-------|----------|
| 1 | 100 |
| 3 | 300 |
| 15 | 500 |
| 25 | 1000 |

**Подсказки:**
1. `w = int(input())`
2. `if w <= 1: ... elif w <= 5: ...`
3. Решение (полный if/elif/else)

### Урок 8: forloop
**Название:** Цикл for и range
