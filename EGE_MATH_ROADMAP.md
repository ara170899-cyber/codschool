## Критические доработки для подготовки на 100 баллов

### 1. ИИ-репетитор по математике
**Приоритет: ВЫСОКИЙ**

Персональный AI-ассистент, специализированный на ЕГЭ математике:
- Объясняет решение шаг за шагом
- Задаёт наводящие вопросы, не давая готовый ответ
- Знает слабые места ученика и предлагает практику
- Может разобрать ЛЮБУЮ задачу (даже загруженную фото — будущее)

```
System prompt:
Ты — репетитор по математике, готовишь к ЕГЭ 2026.
Ученик решает задание №{taskNumber}: {title}
Задача: {description}
Ответ ученика: {userAnswer} (правильный: {correctAnswer})

ПРАВИЛА:
- Если ученик ошибся — НЕ давай ответ сразу
- Спроси: "Какую формулу ты использовал?"
- Покажи где ошибка в рассуждении
- Дай подсказку, которая направит к ответу
- Если ученик совсем не понимает — объясни теорию с нуля
- Используй аналогии из жизни
- Пиши формулы текстом: sqrt(x), x^2, pi
```

**Файлы:**
- `src/lib/math-ai.ts` — AI-ассистент для математики
- `src/app/api/math-assistant/route.ts` — API endpoint
- `src/components/math-ai-chat.tsx` — Чат в Math Player

- Авторские задачи

### 3. Пошаговый решатель
**Приоритет: ВЫСОКИЙ**

Не просто "ответ: 5", а интерактивный разбор:
```
Задача: Найдите площадь треугольника с основанием 10 и высотой 6.

Шаг 1: Вспомним формулу площади треугольника
        S = ½ · a · h
        [Понятно ✓] [Объясни подробнее]

Шаг 2: Подставим значения
        S = ½ · 10 · 6
        [Понятно ✓]

Шаг 3: Вычислим
        S = ½ · 60 = 30
        [Понятно ✓]

Ответ: 30
```

Ученик проходит каждый шаг, может попросить объяснение.

**Файлы:**
- `src/types/math.ts` — добавить `steps: SolutionStep[]` в MathTask
- `src/components/step-solver.tsx` — интерактивный решатель

### 4. Умная система повторения (Spaced Repetition)
**Приоритет: СРЕДНИЙ**

Как Anki, но для ЕГЭ:
- Задачи, которые ученик решил неправильно, показываются чаще
- Задачи, которые решил правильно 3 раза подряд — реже
- Алгоритм: если ошибся → через 1 день, если верно → через 3 дня, потом 7, 14, 30

**Файлы:**
- `src/lib/spaced-repetition.ts` — алгоритм SM-2
- `src/app/(dashboard)/course/ege-math/practice/page.tsx` — умная тренировка

### 5. Конструктор формул
**Приоритет: СРЕДНИЙ**

Визуальный редактор формул:
- Дроби: шаблон ▢/▢
- Степень: x▢
- Корень: √▢
- Интеграл, предел (для части 2)
- Рендер через KaTeX

**Файлы:**
- `src/components/formula-builder.tsx` — визуальный конструктор
- Интеграция с Math Player

### 6. Видео-разборы (заглушки)
**Приоритет: СРЕДНИЙ**

Для каждого типа задания — 5-10 минутное видео:
- Теория за 3 минуты
- Разбор типовой задачи
- Лайфхаки и ловушки
- YouTube embed или собственный плеер

### 7. Система ошибок и ловушек
**Приоритет: ВЫСОКИЙ**

Для каждого типа задания — блок "Частые ошибки":
```
❌ Ошибка: Забыл поделить на 2 в формуле площади треугольника
   Написал S = a · h вместо S = ½ · a · h

❌ Ошибка: Перепутал sin и cos
   sin 30° = ½, cos 30° = √3/2 (не наоборот!)

❌ Ошибка: Не проверил ОДЗ в логарифмическом уравнении
   log₂(x-3) = 1 → x = 5, но нужно проверить: x-3 > 0
```

### 8. Адаптивная сложность
**Приоритет: СРЕДНИЙ**

- Начинать с простых задач (★)
- Если 3 подряд правильно → давать сложнее (★★)
- Если ошибся → вернуться к простым
- Показывать прогресс: "Вы на уровне 70 баллов → для 80 нужно освоить: ..."

### 9. Календарь подготовки
**Приоритет: НИЗКИЙ**

- "До ЕГЭ осталось N дней"
- План на каждый день: "Сегодня: 5 задач на вероятность + 1 вариант"
- Уведомления (push/email)
- Трекер выполнения плана

### 10. Справочник формул
**Приоритет: ВЫСОКИЙ**

Быстрый доступ ко всем формулам:
- Геометрия (площади, объёмы, теоремы)
- Алгебра (формулы сокращённого умножения, логарифмы, степени)
- Тригонометрия (значения, формулы приведения)
- Производные (таблица)
- Вероятность

Доступен из любого места: кнопка "📋 Формулы" в sidebar.
